# Quote editor

This is a small project with Turbo Drive (turbo_frames and turbo_stream), you can save quotes, add dates and line items to manage prices.

## Technologies

- Rails 7.0.4
- Ruby 3.1.2
- Bootstrap 5.0
- Postgresql
- Turbo Drive
- CSS
